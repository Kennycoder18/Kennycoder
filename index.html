Web Development
& Design

Web Design is a specialisation of design stream. It deals with the use of relevant softwares, writing code and mark up validators to create design elements. Designing Web applications requires a prior understanding of Code languages e.g HTML(Hyper Text Markup Language), CSS(Cascading Style Sheets), JavaScript and more.. Web developers use I.D.Es(Integrated Development Environment) also known as text editors e.g Visual Studio, VScode, Atom, Sublime text, Vim and more.. Developers are classified into Frontend and Backend depending on the coding languages and Frameworks executed. Frontend works on user interface (HTML,CSS,JavaScript) whereas Backend works on logic and dynamic functionality(C#,C++,SQL,PHP
,Ruby,Python) of the web application.
